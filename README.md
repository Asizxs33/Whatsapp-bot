# Whatsapp-bot

Студенттерге арналған силлабус анықтамалығы (bot + admin panel).

- **Бот логикасы:** `whatsapp-web.js` арқылы WhatsApp-пен байланысады.
- **Admin Panel:** Express.js арқылы ботты және силлабус базасын басқарады.
- **Деректер базасы:** PostgreSQL (Neon)

## Қалай іске қосуға болады?

1. Пакеттерді орнату: `npm install`
2. `.env` файлын толтыру (Neon DB байланысы).
3. Деректер базасын құру: `npm run init-db`
4. Серверді іске қосу: `npm start`
5. Браузерден ашу: `http://localhost:3000`
